# PrairieSpark Field Performance Pilot

This project contains the Business Requirements Document (BRD) and supporting assets
for a field performance & yield stability dashboard built on a 3,500-acre dryland
grain farm in Saskatchewan.

## What’s here

- `Documents/BRD_PrairieSpark_Field_Performance_KahleSalisbury.pdf` – full Business Requirements Document
- `data/` – (coming soon) anonymized field–year–crop dataset
- `analysis/` – (coming soon) KPI calculations / dashboard files

## Skills demonstrated

- Requirements gathering & stakeholder analysis
- KPI design and data modeling for agriculture
- Functional & non-functional requirements for a BI dashboard
